# Simple Webpack Karma Starter Template

This is a simple starter template for working with Webpack and Karma.

## Getting started

Install dependencies

```bash
npm install
```

## Run Tests

```bash
karma start
```

## Webpack Dev Server

```bash
webpack-dev-server
```

Then point your browser to http://localhost:8080
